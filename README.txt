
This is a setup for building command line network utilities for Android.  

--------
Building
--------

First, You need the Android NDK r5 or newer.  Set the NDK_ROOT to
where your Android NDK is installed, then run the script like this:

  NDK_ROOT=/path/to/android-ndk-r5b ./build-for-android.sh

The results will end up in a little UNIX tree in root/
